Docker MYSQL

  • 启动创建容器

  • 进入容器

    • docker exec -it 容器名称或ID bash

  • 等录mysql

    • mysql -u root -p
    • 输入密码

  • 添加远程登录用户

  • 为用户增加权限

  • 刷新用户权限

注意：

  • 如果你的mysql数据库版本是5.5左右的可以不用添加远程连接就可以直接使用root加密码进行连接
  • 如果是docker里面的mysql最新的8.0镜像你就需要添加一个新的远程连接用户因为
    • 最新的mysql的加密插件默认的是caching_sha2_password的加密 插件/规则 Navicat 无法连接
